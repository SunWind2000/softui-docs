Icon 图标 #

提供了一套常用的图标集合。

使用方法 #

直接通过设置name属性为 s-icon-iconName 来使用即可。例如：

vue
<template>
  <s-icon name="s-icon-edit"></s-icon>
  <s-icon name="s-icon-share"></s-icon>
  <s-icon name="s-icon-delete"></s-icon>
</template>
